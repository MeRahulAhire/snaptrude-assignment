### Instruction to run this project locally

- Clone the repo

```bash
git clone https://github.com/MeRahulAhire/snaptrude-assignment.git
```

- Enter the directory

```bash
cd snaptrude-assignment
```

- Install all the dependency

```bash
npm install
```

- Run the project locally by

```bash
npm run dev
```
